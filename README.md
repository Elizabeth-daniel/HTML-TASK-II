# 👩🏾‍💻 Personal Profile Page

A simple personal profile webpage built with HTML. This project introduces basic web development concepts such as structuring content, adding images, and linking external pages.

##  Preview

This page displays:
- A profile image
- A short introduction
- Fun facts about me
- A link to connect on LinkedIn

##  Features

- Clean and simple layout
- Personal introduction section
- Bullet-point list of fun facts
- External link to LinkedIn profile
- Beginner-friendly HTML structure
